Two files ('opencorpora.sent.train.ru' and 'opencorpora.sent.test.ru')in this folder are the split of entire opencorpora dataset. This split is 80/20 for train/test sets for each of sentence type (.?!).
Each line contains single sentence.

The file 'opencorpora.corpus.test.ru' is the file that contains sentences from 'opencorpora.sent.test.ru' that are combined into paragraphs. This file should be used for testing sentence splitter models.
