В данной папке предполагается хранить различные проекты.

Все новые проекты, по крайней мере для JAVA, желательно создавать как maven проекты.
Это можно сделать при помощи следующей команды (которая должа быть выполнена из текущего каталога)
mvn archetype:generate -DgroupId=org.csclub -DartifactId={project-name} -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

Можно использовать любую IDE для работы над проектом. Надо быть внимательным, чтобы не запушить в гитхаб 
файлы и каталоги, специфичные для соответствующей IDE, а также output папки, куда складываются скомпилированные классы.
По идее, ничего кроме исходных файлов и файла POM в github быть не должно.

NetBeans. 
Чтобы открыть maven проект в NetBeans, надо выбрать Open project в меню File. Перейти к папке projects.
Соответствующие проекты будут подсвечены значком 'Ma' (maven projects). Надо выделить этот проект и нажать кнопку
Open Project. NetBeans знает, что это maven project и умеет с ним работать. В частности, он будет сам обновлять файл POM (это хорошо)).
При использовании NetBeans, эта IDE создает в папке проекта папку target. Она и все её содержимое не должно пушиться в репозиторий.

IDEA.
Чтобы работать в IDEA с maven проектами, надо сначала импортировать проект. Выберите Import Project. В дереве папок выберите 
папку с проектом. Нажмите OK. В следующем окне выберите 'Import project from external model' и отметьте 'Maven'. В следующем окне
я ничего не отмечал. Стоит отметить, что у меня уже был настроен на ноуте maven. Далее в следующих окнах скорее всего надо будет просто нажимать
Next. Скорее всего, надо будет только указать путь к JDK. Лучше использовать версию 1.7. IDEA создать подпапку '.idea', которую не надо пушить
в репозиторий! Как работае
